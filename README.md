# Сайт выставки «Синтез: Технологии. Искусство. Знание»

"Синтез" — выставка, которая исследует границы между наукой, технологиями и искусством. Мы живём в эпоху, когда креативные индустрии становятся не просто сопровождением науки, а её партнёром и двигателем. Эта выставка покажет, как технологии становятся медиумом для художественного высказывания, а искусство — инструментом для научного поиска.

## О проекте

Выставка проходит на территории Университета Дубна.
